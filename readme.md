# App-Mines-Cartel-2016
app iphone cartel mines  Paris  2016 


### To run in a browser

First install dependencies : 
    ```shell
        npm install
    ```

    Then

    ```shell
        gulp browsersync
    ```

### To run in a smartphone

Go in dist/app/dev : 
    ```shell
        cordova platform add android
    ```

    Then

    ```shell
        cordova run android
    ```
